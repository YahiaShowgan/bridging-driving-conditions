Technical Overview

Hybrid Pipeline

Combines simulation (CARLA), diffusion (Stable Diffusion/ALDM), and GAN-based adaptation for cost-effective adverse condition generation

Enhanced DA-UNIT

Extended architecture supporting depth, semantic, and instance data for improved object preservation and label alignment

Blending Technique

Novel approach to correct diffusion artifacts while preserving photorealistic enhancements

Performance Gains

+12.1% object detection and up to +4.6% semantic segmentation improvement on ACDC benchmark
